# Influencer Management System

A Flask-based web application for managing influencer marketing campaigns.

## Features
-   **Multi-User Roles**: Brand, Influencer, Admin.
-   **Campaign Management**: Brands can create and manage campaigns.
-   **Ad Requests**: Influencers can apply to campaigns; Brands can accept/reject.
-   **Dashboards**: Customized views for each role.

## Tech Stack
-   **Frontend**: HTML, Bootstrap 5
-   **Backend**: Python (Flask)
-   **Database**: MySQL (using `mysql-connector-python`)
